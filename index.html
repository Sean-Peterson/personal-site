    • Work I'm Most Proud of

    Take Home

    Take Home provides users the ability to calculate how much money they will be taking home after taxes and living expenses based on which city the user lives in. Users can also compare two cities at one time. For example, if a user has a job offer in Los Angeles for 90k and one in San Francisco for 100k then the user can compare the two cities to see in which city they will take home more money. This app was created using symfony2

    Dream Board

    Dream Board is personal travel blog and photosharing service. My girlfriend Morgan and I are planning a trip to Asia and wanted to share our experience with friends and family. So, I decided to create a way to share our experiences. Visitors can filter the results by clicking on a flag. The app uses OAuth for security. Only Morgan and myself are able to create, update, and delete data. Anyone can view the site. Dream Board is an angular2 firebase app.

    CODEpendency

    CODEpendency was created to help all students at Epicodus study for technical interviews. It is a flashcard app that solves the problem of carrying physical flashcards around. CODEpendency uses an algorithim to make questions that a user gets wrong appear more often. All users can view any deck and answer the questions. When the user decides to "flip" the card to see the answer then the users marks if he/she got the answer right. Each question and answer are saved to the logged in user and that each instance puts a weight on which card will appear next. CODEpendency is an angular2 firebase app.
